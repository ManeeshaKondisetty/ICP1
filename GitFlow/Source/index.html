UNDERSTANDING GIT FLOW

CLONE

Will clone the remote repository to local github desktop.


FETCH

Can periodically get updated code from remote repo to local repo.


PULL

Will merge local repository to your working copy which is in progress.


COMMIT

Will add local changes from your working copy to the local repository.


PUSH

Will push the committed code from local repository to remote repository.


PULL REQUEST

Will create a request to merge code from one branch to another.
